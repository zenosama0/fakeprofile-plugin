# FakeProfile

A Kettu/Vendetta plugin for client-side profile modifications.

## Install

Paste this URL into Kettu Plugins: https://zenosama0.github.io/fakeprofile-plugin/

## Features
- Change any user's avatar, banner, display name, bio, status, and pronouns locally
- Support for multiple users via an array of overrides
- Add/remove/edit overrides from plugin settings
