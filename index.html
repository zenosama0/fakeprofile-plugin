FakeProfile

Client-side profile modifications for Discord mobile (Kettu/Bunny/Vendetta)

Paste this URL into your plugin loader:

https://zenosama0.github.io/fakeprofile-plugin/index.js

Features

  • Change any user's avatar, banner, display name, bio, and pronouns locally
  • Set custom status for any user (client-side only)
  • Support for multiple users via add/edit/remove interface
  • All changes are client-side and only visible to you
